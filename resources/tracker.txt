Track commands
====================================================================================================
00             End part or segment
----------------------------------------------------------------------------------------------------
E0 xx          Instrument change
   xx          Instrument ID
----------------------------------------------------------------------------------------------------
E1 xx          Set pan
   xx          lr.ppppp
                 p - level [0,14] => 00 right, 0A middle, 14 left
                 r - phase invert left
                 r - phase invert right
----------------------------------------------------------------------------------------------------
E2 xx yy       Panning gradual change
   xx          Duration
   yy          Target
----------------------------------------------------------------------------------------------------
E3 xx yy zz    Vibrato
   xx          Delay
   yy          Rate
   zz          Magnitude
----------------------------------------------------------------------------------------------------
E4             Disable vibrato
----------------------------------------------------------------------------------------------------
E5 xx          Set master volume
   xx          Volume
----------------------------------------------------------------------------------------------------
E6 xx yy       Master volume gradual change
   xx          Duration
   yy          Target
----------------------------------------------------------------------------------------------------
E7 xx          Set tempo
   xx          Tempo => BPM * 15360 / 60000 (for $30 quarter note)
----------------------------------------------------------------------------------------------------
E8 xx yy       Tempo gradual change
   xx          Duration
   yy          Target
----------------------------------------------------------------------------------------------------
E9 xx          Transpose global
   xx          Signed absolute new transpose
----------------------------------------------------------------------------------------------------
EA xx          Transpose channel
   xx          Signed absolute new transpose
----------------------------------------------------------------------------------------------------
EB xx yy zz    Volume tremolo
   xx          Delay
   yy          Rate
   zz          Magnitude
----------------------------------------------------------------------------------------------------
EC             Disable tremolo
----------------------------------------------------------------------------------------------------
ED xx          Channel volume
   xx          Volume level
----------------------------------------------------------------------------------------------------
EE xx yy       Volume gradual change
   xx          Duration
   yy          Target
----------------------------------------------------------------------------------------------------
EF xx yy zz    Call part subroutine
   xx yy       Pointer $yyxx
   zz          Number of plays
----------------------------------------------------------------------------------------------------
F0 xx          Set vibrato gradient timer
   xx          Duration
----------------------------------------------------------------------------------------------------
F1 xx yy zz    Glissando to
   xx          Delay
   yy          Duration
   zz          Change
----------------------------------------------------------------------------------------------------
F2 xx yy zz    Glissando from
   xx          Delay
   yy          Duration
   zz          Change
----------------------------------------------------------------------------------------------------
F3             Disable glissando
----------------------------------------------------------------------------------------------------
F4 xx          Fine tuning
   xx          Level
----------------------------------------------------------------------------------------------------
F5 xx yy zz    Configure echo
   xx          EON
   yy          Volume left
   zz          Volume right
----------------------------------------------------------------------------------------------------
F6             Disable echo
----------------------------------------------------------------------------------------------------
F7 xx yy zz    Configure echo filter
   xx          Delay
   yy          Feedback
   zz          Parameters index
----------------------------------------------------------------------------------------------------
F8 xx yy zz    Echo gradual change
   xx          Timer
   yy          Left target
   zz          Right target
----------------------------------------------------------------------------------------------------
F9 xx yy zz    Single glissando
   xx          Delay
   yy          Duration
   zz          Target note
----------------------------------------------------------------------------------------------------
FA xx          Set percussion base note
   xx          Base sample
----------------------------------------------------------------------------------------------------
FF xx yy       Loop SFX
   xx yy       Pointer $yyxx
